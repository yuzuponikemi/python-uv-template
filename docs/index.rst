python-uv-template ドキュメント
===================================

Python研究用ソフトウェア開発のテンプレートリポジトリへようこそ。

このテンプレートは、テスト駆動開発（TDD）をベースに、再現性と品質を重視した
研究用Pythonソフトウェア開発をサポートします。

主な特徴
--------

* **uv** による高速な依存関係管理
* **テスト駆動開発** を前提とした構成
* **Claude Code** による自動化されたコード支援
* 自律的なコード修正システム
* 研究用ソフトウェアに最適化された設定
